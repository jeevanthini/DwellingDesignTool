 🏗️ ArchiGen AI - Architectural Design Platform

 🌟 Overview

ArchiGen AI is a revolutionary web application that transforms architectural design through artificial intelligence. Create professional 2D floor plans and immersive 3D visualizations with ease, making architectural design accessible to everyone.

 ✨ Features

 🎨 Design & Visualization
- AI-Powered Floor Plans - Intelligent room layout generation
- 2D/3D Toggle - Switch between professional floor plans and immersive 3D views
- Real-time Editing - Drag-and-drop room customization
- FPS Walkthrough - First-person exploration of your designs
- Day/Night Mode - Visualize your space in different lighting conditions

 📐 Professional Tools
- Multiple Architectural Styles - Modern, Classic, Contemporary, Traditional, Minimalist, Luxury, Eco-friendly
- Various House Types - Apartment, Villa, Duplex, Bungalow, Mansion
- Precise Measurements - Accurate dimensioning and area calculations
- Room Type Management - Living rooms, bedrooms, kitchens, bathrooms, and more

 💾 Export & Sharing
- Multi-Format Downloads - PDF, PNG, JPG, OBJ formats
- Quality Settings - Ultra HD to optimized web formats
- Print Ready - Professional presentation documents
- Share Projects - Secure link sharing with clients and team

 🎭 Immersive Experience
- Animated Construction Theme - Building process visualization
- Glass Morphism UI - Modern, professional interface
- Interactive 3D Environment - Real-time rendering and exploration
- Storytelling Elements - Hard work, smart work, and passion themes

 🚀 Quick Start

 Prerequisites
- Node.js (v14 or higher)
- npm or yarn

 Installation

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/archigen-ai.git
   cd archigen-ai
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

4. Open your browser
   ```
   http://localhost:5173
   ```

 Building for Production

```bash
npm run build
npm run preview
```
